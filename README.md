# linux-essentials-labs

A hands-on guide to mastering Linux essentials. This repository includes practical exercises, command explanations, and mini-projects to help beginners and enthusiasts strengthen their Linux skills through real-world scenarios

Following topics are covered as part of the course and documentation.

- Fundamentals of Linux
- Linux vs Windows
- Core components of Linux
- Setup Linux on Windows & MacOS
- Linux folder structure
- Linux user management
- Linux file management
- VI Editor shortcuts (commonly used)
- File permissions
- Process management
- Linux system monitoring
- Basic Networking in Linux
- Disk and Storage management in Linux

Please refer to the folders at the root level of this repository to go through the documentation.
